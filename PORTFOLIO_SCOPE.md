# Voithos Portfolio Scope

Este diretorio e uma copia sanitizada da Voithos para apresentacao profissional.

Objetivo:
- mostrar arquitetura, organizacao e capacidade de entrega
- evitar exposicao de dados, segredos e detalhes operacionais
- facilitar avaliacao tecnica por recrutadores

Recorte inicial recomendado:
- agenda e fluxo de agendamentos
- autenticacao e perfis
- comunicacao e integracao com WhatsApp
- financeiro e gestao
- arquitetura Electron + backend + servicos compartilhados

O que nao entra no portfolio publico:
- arquivos .env reais
- logs e arquivos de runtime
- banco local e dados persistidos
- recibos, pacientes e registros operacionais
- segredos e tokens

- criar README de apresentacao
- deixar setup orientado a demonstracao
- incluir dados ficticios e narrativa tecnica
